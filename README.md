# Bolt Network React

An app for searching and viewing movie data from IMDB with an
interface that mimics the UI and features of Netflix.

[Live Site Link](https://lit-anchorage-35230.herokuapp.com/)

Built with:
* Ruby 2.5.0
* Ruby on Rails 5.2.1
* Webpacker 4.2.0
* PostgreSQL 11
* React.js 16.12.0
